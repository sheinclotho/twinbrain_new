# TwinBrain V5 - 图原生数字孪生脑训练系统

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install torch torch-geometric nibabel mne pyyaml numpy scipy
```

### 2. 配置数据路径
编辑 `configs/default.yaml`:
```yaml
data:
  root_dir: "/path/to/your/data"  # 修改为你的数据路径
```

### 3. 运行训练
```bash
python main.py
```
